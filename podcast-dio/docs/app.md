📱 Sobre a Aplicação

O Podcast Manager é uma aplicação back-end desenvolvida para gerenciar informações relacionadas a podcasts.

A aplicação permite organizar conteúdos de forma simples e estruturada, simulando funcionalidades comuns encontradas em plataformas de streaming e gerenciamento de mídia.

⚙️ Funcionalidades
Listagem de podcasts
Organização de episódios
Estruturação de rotas
Retorno de dados em JSON
Separação entre controller, service e routes

🎯 Finalidade

O principal objetivo deste projeto é praticar conceitos fundamentais de desenvolvimento back-end utilizando Node.js e TypeScript, aplicando arquitetura organizada e boas práticas de código.
