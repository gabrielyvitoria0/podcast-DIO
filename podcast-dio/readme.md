# 🎙️ Podcast Manager - DIO

Projeto desenvolvido durante os estudos na plataforma DIO, acompanhado pelo professor Felipe Aguiar.

A proposta do projeto é simular um gerenciador de podcasts, trabalhando conceitos fundamentais de desenvolvimento back-end, organização de rotas e manipulação de dados.

---

## 🚀 Tecnologias Utilizadas

* Node.js
* TypeScript
* Express
* TSX
* Git & GitHub

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

* Estruturação de APIs
* Organização de pastas
* Criação de rotas
* Manipulação de requisições HTTP
* Conceitos de back-end
* Versionamento com Git e GitHub

---

## 💻 Objetivo do Projeto

Este projeto foi criado com foco educacional através da plataforma DIO, buscando fortalecer conhecimentos práticos em desenvolvimento back-end e boas práticas de programação.
